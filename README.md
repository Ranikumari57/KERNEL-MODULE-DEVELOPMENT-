## 📌 Project Title

Character Device Driver Kernel Module (Read/Write Device)

This project creates a virtual device file in Linux like /dev/mydevice where you can write and read data using a kernel module.

## 🎯 Objective

- Create a Linux kernel module
- Register a character device
- Create /dev interface
- Perform read & write operations
- Understand user ↔ kernel communication
  
## ⚙️ Requirements

- Linux OS (Ubuntu recommended)
- GCC compiler
- Kernel headers installed
- Bash
  sudo apt update
 sudo apt install build-essential linuxheaders-$(uname -r)

## 🧩 Features of This Project

- Creates virtual device in /dev
- Supports read/write operations
- Kernel-user communication
- Basic driver development structure
